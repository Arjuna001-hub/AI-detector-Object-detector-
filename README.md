## 🎯 Overview

**Smooth AI Object Detector** is a cutting-edge web application that uses artificial intelligence to identify objects in real-time through your camera. With smooth 60fps performance, voice feedback, and an intuitive interface, it makes object recognition accessible to everyone.

**Try it live:** Simply open the HTML file in any modern browser - no installation required!

---

## ✨ Features

### Core Capabilities
- 🎥 **Real-time Detection** - Identifies 90+ common objects instantly
- 🗣️ **Voice Feedback** - AI speaks object names with detailed descriptions
- 📱 **Responsive Design** - Works perfectly on desktop, tablet, and mobile
- 🚀 **Smooth 60fps** - Optimized rendering for butter-smooth performance
- 🎨 **Visual Feedback** - Color-coded bounding boxes with confidence scores

### Advanced Features
- 🔄 **Auto-Detect Mode** - Continuously analyzes and describes objects
- 🎯 **Click-to-Learn** - Tap any detected object for detailed information
- 📊 **Confidence Metrics** - Real-time confidence percentage display
- 🌈 **HSL Color Coding** - Unique colors for different object types
- 💾 **Zero Data Upload** - Everything runs locally in your browser

### Performance Optimizations
- ⚡ **Throttled Detection** - Optimal 150ms detection intervals
- 🎬 **RequestAnimationFrame** - Smooth 60fps video rendering
- 📦 **Batch DOM Updates** - Efficient UI rendering
- 🎨 **GPU Acceleration** - Hardware-accelerated canvas drawing
- 🔋 **Low Resource Usage** - MobileNet V2 for lightweight processing
